## Mini R Compiler

This project aims to build a ***mini compiler for R language***

 - ABED Abir

The front-end of the compiler consisting of the following four phases is built:

 1. Lexical analysis
 2.  Syntax Analysis 
 3. Semantic Analysis
 4. Intermediate Code Generation
